## Technologies Used

- **Frontend:** HTML, CSS, JavaScript, React
- **Hosting:** Vercel

## Installation & Setup

To run this project locally:

```bash
git clone https://github.com/iamhimanshu98/My-Portfolio.git
cd My-Portfolio
npm install
npm run dev
```
